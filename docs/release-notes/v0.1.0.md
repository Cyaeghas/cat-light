# cat-light v0.1.0

First public prototype release.

## Highlights

- Local-first Codex / Claude Code status monitor.
- CLI commands for quota, state, sessions, history, hooks, sync, and doctor checks.
- Passive local JSONL scanning for Codex sessions and Claude Code projects.
- Reversible hook install/uninstall for Codex `notify` and Claude `settings.json`.
- Local dashboard served on `127.0.0.1`.
- Windows tray launcher.
- Windows floating monitor for active Codex / Claude Code state.
- JSONL history backend with optional bundled SQLite build.
- Windows, Linux, and macOS CI builds.

## Windows Archive

The Windows zip contains:

```text
cat-light.exe
cat-light-tray.exe
cat-light-float.exe
README.md
README.zh-CN.md
docs/install-windows.md
```

Start the floating monitor:

```powershell
.\cat-light-float.exe
```

Open the local dashboard:

```text
http://127.0.0.1:8750
```

## Notes

This is still an early prototype. Provider quota endpoints and local log formats can change, so the parser and quota handling will continue to evolve.
